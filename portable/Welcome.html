🦞 U-Claw

Portable AI Assistant · 3 steps to get started
Quick Start
Step 1: Start the Service
Double-click the launcher for your system:

Windows users: Double-click Windows-Start.bat
Mac users: Double-click Mac-Start.command
Step 2: Configure AI Model
First run auto-opens the config page. Follow the prompts:
1. Select an AI model (recommended: DeepSeek — cheap & powerful)
2. Enter your API Key (register at the model's website)
3. Click "Save Configuration"
Step 3: Start Chatting
After configuration, the browser auto-opens the AI chat interface
URL: http://127.0.0.1:18789
You can now chat with your AI assistant!

✅ All configuration and chat history is stored in the USB's data/ directory, automatically shared between Mac and Windows!

Troubleshooting?
💡 Run Diagnostics Tool

If startup fails, double-click the diagnostics tool:
Windows: Windows-Diagnose.bat
Mac: Mac-Diagnose.command
